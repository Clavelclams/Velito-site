## Procédure rapide

1. Pixabay = la source la plus simple (licence Pixabay = OK commercial)
2. Redimensionne à 800×600 (Squoosh.app fait ça en ligne en 10s)
3. Renomme avec le `id` exact ci-dessus
4. Glisse dans ce dossier
5. Commit + push

Pas besoin de toutes les avoir d'un coup. Le jeu marche avec emoji fallback.
